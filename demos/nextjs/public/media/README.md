# Demo media

The video and poster files are short, silent derivatives of **Big Buck Bunny**
by the Blender Foundation and are redistributable under
[Creative Commons Attribution 3.0](https://creativecommons.org/licenses/by/3.0/).

- Original: https://download.blender.org/peach/bigbuckbunny_movies/BigBuckBunny_320x180.mp4
- Project and credits: https://peach.blender.org/
- Attribution: `(c) copyright 2008, Blender Foundation / www.bigbuckbunny.org`

`bunny-4k.mp4` is an upscaled one-second fixture for exercising a real 4K
decode surface; it is not a native-4K quality sample.

The excerpts are cropped, resized, recompressed, and stripped of their original
audio. `yumcut-demo-music.ogg` is an original synthetic three-tone test bed
generated by `scripts/download-test-media.sh`; it contains no third-party
soundtrack. See the repository-root `MEDIA_LICENSES.md` for full details.
