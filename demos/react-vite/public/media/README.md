# Demo media attribution

The three videos and poster in this directory contain short, recompressed excerpts
from **Big Buck Bunny** by the Blender Foundation, licensed under
[Creative Commons Attribution 3.0](https://creativecommons.org/licenses/by/3.0/).

- Original: https://download.blender.org/peach/bigbuckbunny_movies/BigBuckBunny_320x180.mp4
- Project and credits: https://peach.blender.org/
- Attribution: `(c) copyright 2008, Blender Foundation / www.bigbuckbunny.org`

The excerpts are cropped, resized, recompressed, and stripped of their original
audio. `yumcut-demo-music.ogg` is a synthetic three-tone test track generated by
the repository fixture script and contains no third-party soundtrack.

These files are copied from `test/fixtures/media` so the React demo is standalone
at runtime. Reproduction details and SHA-256 checksums live in
`test/fixtures/README.md` and `test/fixtures/manifest.json` at the repository root.
